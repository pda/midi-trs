pda / midi-trs
==============

[TRS MIDI](https://minimidi.world/) is the future. The future is already here,
it's just not evenly distributed. This is my attempt to improve that.

`midi-solutions-quadda-thru-trs`
--------------------------------

Converting [MIDI Solutions MultiVoltage Quadra Thru](https://www.midisolutions.com/prodqth.htm) from DIN5 to TRS,
with switchable TRS type A/B input, rehoused in painted die-cast aluminium enclosure.


`korg-volca-trs-midi`
-----------------------------------

Replacing the MIDI OUT DIN5 jack with TRS type A for Korg Volca Keys/Beats/Bass etc.
